    Conexão WhatsApp

    Conecte o WhatsApp da sua igreja via Meta Cloud API Oficial para envio de mensagens, automações e Chat ao Vivo.

    Desconectado
    —
    WABA: —
    📋

    Guia de Configuração

    Siga os passos abaixo para configurar tudo corretamente com a Meta

    1 Criar Conta Meta Business ›

    Para usar a API oficial do WhatsApp, sua igreja precisa ter uma conta no Meta Business Suite.

    • Acesse business.facebook.com e crie sua conta empresarial (é gratuito)
    • Use o nome oficial da sua igreja ou organização
    • Adicione um e-mail profissional (evite @gmail.com se possível)
    💡 Dica: Se sua igreja já possui uma Página do Facebook, ela provavelmente já tem uma conta Meta Business.
    2 Verificação do Negócio ›

    A Meta exige que organizações sejam verificadas antes de usar a API do WhatsApp. Prepare os seguintes documentos:

    • 📄 Documento de registro legal da organização (CNPJ, Articles of Incorporation, etc.)
    • 📄 Comprovante de endereço (conta de utilidade ou extrato bancário)
    • 🌐 Website ativo com nome, endereço e informações de contato da igreja

    Acesse Meta Business Suite → Configurações → Central de Segurança → Verificação do Negócio para iniciar.

    ⏱ Prazo: A verificação costuma levar 2-5 dias úteis. Você pode conectar o WhatsApp antes, mas envios em massa só são liberados após a verificação.
    3 Número de WhatsApp Dedicado ›

    Você precisará de um número de telefone dedicado para a API do WhatsApp.

    • O número não pode estar ativo no WhatsApp pessoal ou WhatsApp Business App
    • Se já tiver um número no WhatsApp Business App, será necessário deletar a conta do app para liberar o número para a API
    • Aceita números de celular, fixo ou até VoIP (desde que receba SMS ou chamada)
    📱 Recomendação: Use um número exclusivo da igreja que será reconhecido pelos membros. Não use seu número pessoal.
    4 Conectar via Facebook ›

    Clique no botão "Continuar com Facebook Business" acima para iniciar o processo automatizado de conexão:

    1. Faça login com a conta do Facebook que administra o Meta Business Suite da igreja
    2. Selecione (ou crie) a conta WhatsApp Business Account (WABA)
    3. Vincule o número de telefone dedicado
    4. Pronto — o Zelo salva as credenciais automaticamente!
    ✅ Nenhuma senha é armazenada — o Zelo usa apenas tokens OAuth da Meta, que podem ser revogados a qualquer momento.
    5 Configuração Manual (Avançado) ›

    Se preferir configurar manualmente (para desenvolvedores ou cenários avançados), preencha os campos abaixo com os dados do Meta Developer Console:

    🔗

    Webhook da Meta

    Use estes dados no Meta Developer Console → WhatsApp → Configuração

    Como configurar: No Meta Developer Console, vá em seu App → WhatsApp → Configuração → Webhook. Cole a URL de Callback e o Verify Token acima. Assine os campos: messages.

    🤖 Respostas por IA (Inbound)

    Quando ativado, mensagens recebidas pelo WhatsApp são automaticamente respondidas pelo LLM com base nas instruções e base de conhecimento abaixo.
    ⚠️ A regra de Human Lock (30 min) sempre tem prioridade sobre a IA.

    🧠 Memória de Conversa

    Quando ativado, o LLM recebe o histórico recente da conversa com o lead (últimas 10 mensagens) como contexto antes de responder.

    📋 Instruções do Assistente (System Prompt)

    Define o comportamento, tom e limites da IA. Seja específico sobre o que a IA pode e não pode responder.

    📚 Base de Conhecimento da Igreja

    Informações que a IA usa para responder perguntas dos leads. A mesma base usada pela Mila (suporte interno).

    🗺️ Arquitetura do Agente (Ju)

    Visão lógica de como a Ju processa dados, contexto de tempo e áudio na Zelo Pro.

    graph TD
    User((Lead / Usuário)) -- "Mensagem (Texto/Áudio)" --> WAHA[Webhook WhatsApp]
    WAHA -- "Filtro Heurístico" --> AutoReply{É Auto-Reply?}
    AutoReply -- Sim --> Ignore[Ignorar Mensagem]
    AutoReply -- Não --> proc[Processador]
    
    proc -- "Verifica Human Lock (30m)" --> Lock{Lock Manual Ativo?}
    Lock -- Sim --> Halt[Parar IA]
    Lock -- Não --> Agent[IA Ju Gemini 1.5 Pro]
    
    Agent -. "Lê Base, Tags & Data/Hora" .-> KB[(Contexto Zelo)]
    Agent -. "Usa Ferramenta Whisper" .-> Whisper[Transcreve Áudio]
    
    Agent -- "Decide Ação" --> Tools{Ferramentas}
    Tools -- "Aviso Data USD/BR" --> task[Criar Tarefa CRIE/InPeace]
    Tools -- "Muda Funil/Tag" --> update[Atualiza DB Supabase]
    Tools -- "Requer Humano" --> handoff[Ativa Human Lock]
    
    Tools -- "Resposta Pronta" --> Gen{Geração}
    Gen -- "Texto" --> msg[WhatsApp Texto via Edge]
    Gen -- "Voz" --> audio[WhatsApp Áudio TTS]
    msg --> User
    audio --> User
